Due to the large number of objects, this list includes only the following:
  • All objects up to 10 bits
  • All objects buildable from four gliders or less
  • All non-factorable named objects up to 15 bits
  • All non-factorable objects above 15 bits with constructions shown here
  • Some larger objects with long-standing traditional names.


It typically does not include the extreme proliferation of larger objects:
  • Objects referenced only by number (i.e. 12.121, 12-bit flip-flop #1)
  • Objects which can be factored into several separately-named similar parts (i.e. block on cigar, tripole tie ship, up bipole on table)
  • Trivial expansions of smaller objects (i.e. fumarole w/tub, eater eating integral)
